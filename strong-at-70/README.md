# Strong at 70

Static GitHub Pages app.

Copy this folder into your repo as:

`apps/strong-at-70/`

Then add this card/link to `apps/index.html`:

```html
<a class="app-card" href="strong-at-70/">
  <div class="icon">💪</div>
  <h3>Strong at 70</h3>
  <p>Hip-friendly strength program with video demos.</p>
</a>
```

No build step required. Checkboxes use browser localStorage.
